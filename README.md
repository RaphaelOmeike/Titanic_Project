# Titanic Survival Prediction System

A Machine Learning powered Web GUI to predict Titanic passenger survival.

## Project Structure
- `app.py`: The main Streamlit web application.
- `model/`: Contains the trained model (`titanic_model.joblib`) and the training notebook (`model_building.ipynb`).
- `requirements.txt`: List of Python dependencies.

## How to Run Locally

1.  **Install Dependencies**:
    Open your terminal in this directory and run:
    ```bash
    pip install -r requirements.txt
    ```

2.  **Run the App**:
    Execute the following command:
    ```bash
    streamlit run app.py
    ```

3.  **View**:
    The app will open in your default web browser (usually at `http://localhost:8501`).
